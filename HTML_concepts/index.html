HTML

- Hypertext markup language.

- It is the code used to structure a webpage and its content.

- The component used to design the structure of websites are called Html tags


Basic syntax


        <!DOCTYPE html>

        <html lang="en">

        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">

            <title>Document</title>
        </head>

        <body>

            <h1>HEADING</h1>
            <p>This is paragraph</p>

        </body>

        </html>

Quick Points

- Html tag is parent of head and body tag

- Most of HTML elements have opening and closing tags with cotent in between

- Some tags have no content in between eg. - br tag

- We can use inspect element/view page source to edit html and see html of a webpage


HTML Tags

A container for some content or other HTML Tags

Tags are not case sensitive


HTML Attributes

HTML attributes provide additional information about HTML elements.

- All HTML elements can have attributes

- Attributes provide additional information about elements

- Attributes are always specified in the start tag

- Attributes usually come in name/value pairs like: name="value"

eg. -> href, src, alt, style, lang, title

Anchor Tag

Used to add links to your page

        <a href="https://google.com/">Visit google.com!</a>

This is what it will show:

Visit google.com

Attribute Purpose Example Values
href Specifies link destination "https://google.com", "about.html", "#section"
target Specifies where link opens "_self", "_blank", "_parent", "_top"
title Shows tooltip text on hover "Visit Google", "Open Page"

Image Tag

used to add images to your page

        <img src="../images/bdd35582-9ada-4585-9807-de78d5ffcf6b.jpeg" alt="Image of Tanjiro">

This is what it will show:



Attribute Purpose Example Values
src Specifies image path/source "image.jpg", "../images/photo.png"
alt Alternative text if image cannot load "Nature Image", "Profile Photo"
height Sets image height "200px", "300"
width Sets image width "400px", "500"

HTML

- Hypertext markup language.

- It is the code used to structure a webpage and its content.

- The component used to design the structure of websites are called Html tags


Basic syntax


        <!DOCTYPE html>

        <html lang="en">

        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">

            <title>Document</title>
        </head>

        <body>

            <h1>HEADING</h1>
            <p>This is paragraph</p>

        </body>

        </html>

Quick Points

- Html tag is parent of head and body tag

- Most of HTML elements have opening and closing tags with cotent in between

- Some tags have no content in between eg. - br tag

- We can use inspect element/view page source to edit html and see html of a webpage


HTML Tags

A container for some content or other HTML Tags

Tags are not case sensitive


HTML Attributes

HTML attributes provide additional information about HTML elements.

- All HTML elements can have attributes

- Attributes provide additional information about elements

- Attributes are always specified in the start tag

- Attributes usually come in name/value pairs like: name="value"

eg. -> href, src, alt, style, lang, title

Subscript displays text slightly below the normal line.

Page Layout Techniques

Using semantic tags for layout - using the right tags

This goes inside the body tag

    1. <header>

        Used for:
         - Website title
         - Logo
         - Navigation menu
         - Introductory content

        eg.- <header>
                <h1>My Website</h1>
                <nav>
                    <a href="#">Home</a>
                    <a href="#">About</a>
                </nav>
             </header>


    2. <main>

        Used for:
         - Main content of webpage
         - Important page information
         - Central content area

        eg.- <main>
                <h2>Welcome</h2>
                <p>This is the main content.</p>
             </main>


    3. <section>

        Used for:
         - Grouping related content
         - Dividing webpage into sections
         - Organizing topics

        eg.- <section>
                <h2>Services</h2>
                <p>We provide web development services.</p>
             </section>


    4. <article>

        Used for:
         - Blog posts
         - News articles
         - Independent content

        eg.- <article>
                <h2>Latest News</h2>
                <p>This is a news article.</p>
             </article>


    5. <aside>

        Used for:
         - Sidebar content
         - Related links
         - Additional information

        eg.- <aside>
                <h3>Related Links</h3>
                <p>Extra information here.</p>
             </aside>


    6. <footer>

        Used for:
         - Copyright information
         - Contact details
         - Social media links

        eg.- <footer>
                <p>© 2026 My Website</p>
             </footer>

Semantic Tags

Semantic tags are HTML tags that clearly describe the meaning of the content they contain. They tell: Browser, Developer, Search engine - what type of content is inside the tag. Semantic tags improve: Readability SEO Accessibility Code maintenance

Eg.- footer, header, aside, article, section, main

Non semantic tags

Non-semantic tags are HTML tags that do not describe the meaning of the content inside them. They only help in: Structuring content, Grouping elements, Styling with CSS, Adding functionality with JavaScript

These tags do not tell the browser or developer what type of content they contain. 

    1. <div>

        Used for:
         - Grouping HTML elements
         - Creating webpage sections
         - Styling using CSS
         - Layout designing

        It is a block-level element.

        eg.- <div>
                <h2>Welcome</h2>
                <p>This is inside a div tag.</p>
             </div>



    2. <span>

        Used for:
         - Styling small parts of text
         - Grouping inline content
         - Applying CSS or JavaScript

        It is an inline element.

        eg.- <p>
                This is 
                <span>important</span> 
                text.
             </p>

Difference between Semantic and Non Semantic tags

Semantic Tags Non-Semantic Tags
Describe meaning of content Do not describe meaning
Examples: <header>, <footer> Examples: <div>, <span>
Better for SEO and accessibility Mainly for styling/layout
Easier to understand Generic containers

Lists

Lists in HTML are used to display items in a structured format. There are mainly two types of lists: Ordered and Unordered List 


    1. Ordered List (<ol>)

        Used for:
         - Numbered items
         - Step-by-step instructions
         - Sequential data

        Tags used:
         - <ol> : Ordered list
         - <li> : List item

        eg.- <ol>
                <li>Wake up</li>
                <li>Brush teeth</li>
                <li>Go to college</li>
             </ol>

This is what it will show:

  1. Wake up
  2. Brush teeth
  3. Go to college

    2. Unordered List (<ul>)

        Used for:
         - Bullet point items
         - Non-sequential data
         - Grouping related items

        Tags used:
         - <ul> : Unordered list
         - <li> : List item

        eg.- <ul>
                <li>Apple</li>
                <li>Banana</li>
                <li>Mango</li>
             </ul>

This is what it will show:

  • Apple
  • Banana
  • Mango
